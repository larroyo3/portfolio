# Lucas Arroyo - Portfolio Mobile Expert

Ce projet est un site portfolio one-page ultra-performant conçu pour Lucas Arroyo, expert en développement mobile (Kotlin Multiplatform, Jetpack Compose, SwiftUI).

## 🚀 Technologies Utilisées
- **React 18** + **TypeScript**
- **Vite** (Build tool ultra-rapide)
- **Tailwind CSS** (Design system basé sur des tokens)
- **Framer Motion** (Micro-interactions fluides)
- **Lucide React** (Iconographie moderne)

## 🎨 Design System
Le design utilise une palette de couleurs spécifique (Radix Colors inspired) pour les modes clair et sombre.
- **Tokens** : `/src/styles/tokens.json`
- **Styles Globaux** : `/src/index.css`

## 🛠️ Installation & Développement
1. Cloner le repo
2. Installer les dépendances :
   ```bash
   npm install
   ```
3. Lancer le serveur de dev :
   ```bash
   npm run dev
   ```
4. Build pour la production :
   ```bash
   npm run build
   ```

## 📦 Déploiement
Le projet est prêt à être déployé sur **Netlify** ou **Vercel**. 
Il suffit de lier votre dépôt GitHub et d'utiliser la commande `npm run build` avec le dossier de sortie `dist`.

## 📝 Personnalisation
Pour modifier le contenu (projets, expériences, services), éditez les composants correspondants dans `src/components/`.
Les images peuvent être remplacées dans `src/assets/` ou via des URLs dans les données des composants.
